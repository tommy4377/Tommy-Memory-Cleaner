# Tommy Memory Cleaner - Installer Personalizzato

## Installazione Inno Setup

Per compilare l'installer personalizzato, è necessario:

1. Scaricare e installare [Inno Setup](https://jrsoftware.org/isdl.php) (versione 6.0 o superiore)
2. Aprire `installer.iss` con Inno Setup Compiler
3. Compilare lo script per generare l'installer

## Struttura File

- `installer.iss` - Script principale Inno Setup
- `welcome.html` - Interfaccia HTML personalizzata (opzionale, per uso futuro)
- `output/` - Directory dove verrà generato l'installer

## Caratteristiche Installer

- ✅ Interfaccia personalizzata con stile dell'app
- ✅ Mini tutorial delle funzionalità
- ✅ Selezione lingua (6 lingue supportate)
